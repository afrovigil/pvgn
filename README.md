# Assets Publics (React / Vite)

Placez vos fichiers d'images ici (par exemple `notif.pdf` et `dgpml.png`).
Vite copiera automatiquement tous les fichiers de ce dossier vers la racine du dossier de build `dist/` lors de la compilation (`npm run build`).

Ainsi :
- `public/notif.pdf` sera accessible via `src="notif.pdf"` dans l'application React.
- `public/dgpml.png` sera accessible via `src="dgpml.png"`.
